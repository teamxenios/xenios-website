# Shop by Goal

The primary navigation a member uses. A member arrives with a goal in their own words, not with
a compound name, so the goal page is the front door to everything else.

## Navigation terminology rule

Primary navigation uses plain member language, never clinical or industry framing:

| Use this | Never use this in navigation |
|---|---|
| Think Sharper | Cognitive Function |
| Everyday Health | Foundational Health |
| Intimacy and Vitality | Hormone Support |

## Page structure

Every goal page carries these sections in order:

1. Plain-language definition
2. Lifestyle foundation
3. Fitness and nutrition considerations
4. Supplement Foundation categories (category level only, never a branded SKU)
5. Optional research-education pathways (education only, never a recommendation to obtain)
6. What Xenios will not promise
7. Related Guides
8. Assessment call to action

## The two sections that matter most

**Lifestyle foundation** and **What Xenios will not promise** are the trust anchors. A member
should finish a goal page trusting Xenios more, not wanting to buy something. If a page reads
as persuasion, it has failed and should be rewritten.

Goal mappings are a navigation aid based on what a compound has been studied for. A mapping is
never a claim that the compound produces the goal.

## Constraints

- Supplement Foundation is category level only. No branded SKU appears, because no reseller
  authorization exists.
- Research-education pathways are educational. Peptide commerce is not authorized, and every
  page says so.
- The assessment is what makes anything personalized. It is not a diagnosis and it is not a
  prescription.
