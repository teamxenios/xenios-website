Replit Prompt — Xenios Coming Soon / Waitlist (one page)
Goal: Replace the entire current xeniostechnology.com homepage with a single, calm, magazine-grade "coming soon + waitlist" page. No nav, no scrolling sections, no feature lists. One screen. One job: capture the email.
The new page should pull design DNA from the attached Particular letterhead reference — a Tishman Speyer / BIG / Production / Design & Engineering Venn — and translate that aesthetic into Xenios's existing visual world.

1. The design reference (what to copy structurally)
The Particular letterhead works because of five moves. Use all five:

Three overlapping pale circles forming a Venn, centered on the page. Very low opacity (~12–18%). The circle strokes/fills are soft, almost watermark.
A label sits in the center of each circle and in the center of the intersection. Center label is the brand name — the thing the three forces converge into.
Wordmark top-left, tiny metadata top-right (a date, an issue number, a status — typeset in monospace at ~10px, uppercase, letter-spaced).
A single short paragraph of "manifesto" copy in the lower-left corner, monospace, tight, ~9–10px.
Tiny corner anchors — page number / location / vertical line of running text on the right margin. Generous, almost wasteful whitespace everywhere else.

Do not copy the literal blue circles or the literal Particular labels. Translate the structure into Xenios.

2. Colors (lock to these — pulled from current site + pitch deck)
Use a light/white page (not the deck's black — the public site is white, keep that). But use the deck's accent gradient as the one moment of color:
--bg:          #FFFFFF   /* paper white */
--ink:         #0A0A0A   /* near-black for type */
--ink-soft:    #6B6B6B   /* metadata, captions */
--rule:        #E5E5E5   /* hairlines */

/* The Xenios gradient — same one used on the current homepage
   on the "functional medicine doctors" pill. This is the ONLY
   color on the page. Use it for the circles and the submit button. */
--grad-from:   #7C3AED   /* violet */
--grad-via:    #6366F1   /* indigo */
--grad-to:     #06B6D4   /* cyan */
The three overlapping circles each get the gradient, but at very low opacity (12–15%) and mix-blend-mode: multiply so where they overlap the color deepens naturally. That deepening is the visual — it's the whole point.

3. Typography

Headlines / wordmark: a clean grotesque. Use Inter (already in the stack) or Söhne if available. Tight tracking on the wordmark.
The three circle labels + center label: same grotesque, all lowercase, weight 500, ~22–28px on desktop.
All metadata, status pills, manifesto paragraph, footer: JetBrains Mono or IBM Plex Mono, uppercase, 10px, letter-spacing 0.12em, color --ink-soft.

One typeface family for content, one mono for chrome. Nothing else.

4. Page layout (one screen, no scroll on desktop ≥1024px)
A single min-h-screen flex layout with these zones, exactly like the Particular sheet:
Top bar (horizontal, ~64px tall, hairline rule beneath)

Left: xenios. wordmark (lowercase, period included, weight 600)
Right: [● LIVE] FOUNDING COHORT · MAY 2026 in mono, 10px uppercase. The ● dot is filled with the gradient and gently pulses (2s ease-in-out infinite).

Center stage (takes the full remaining height, flex-centered)
The Venn. Three circles, each ~360px diameter on desktop, arranged in the classic three-circle Venn (two on top, one centered below — or one top, two below, your call; mirror the Particular layout).

Left circle label (centered inside the circle, above the intersections): coaches
Right circle label: trainers
Bottom circle label: practitioners
Center label (where all three overlap): the word xenios at ~32px, weight 600, color --ink. This is the visual punchline — three professions, one operating system.

Each circle: gradient fill at 14% opacity, no stroke, mix-blend-mode: multiply. Subtle 20s slow drift animation (each circle translates ±6px on a different phase) so the page breathes. Respect prefers-reduced-motion.
Headline + waitlist (sits below the Venn, still inside the viewport)

7. The feel — say this back to yourself before shipping
It should feel like a piece of stationery from an architecture firm. Almost embarrassingly empty. The gradient appears in exactly two places: the three overlapping circles (faint) and the submit button (saturated). Everything else is black ink on white paper with thin mono labels in the corners. No cards, no shadows, no rounded boxes around things, no gradients on text, no "✨" emojis, no glass morphism. If it looks like a SaaS landing page, it's wrong. If it looks like a museum invitation, it's right.
