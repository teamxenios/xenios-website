# Final Document Dependency Map

| Stage | Required documents or controls |
|---|---|
| Before identity upload | Electronic Records Consent; Privacy Policy; Identity Verification and Government ID Consent |
| Before activation payment instructions | Founding Membership Agreement; Confidentiality NDA; Research Use Agreement; No-Medical-Advice and Assumption-of-Risk; Arbitration Agreement; Release and Indemnification; Manual Payment Terms; Renewal Policy; Cancellation and Refund Policy; Website Terms |
| Before payment-evidence upload | Payment Evidence Upload Consent |
| Before membership activation | Identity review complete; all activation agreements accepted; actual payment receipt verified |
| Before first product order | Product Purchase Terms; Shipping, Claims and Replacement Policy; current Research Use Agreement |
| On material legal revision | Reacceptance according to the document version's publication rule |
